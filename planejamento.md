## Objetivo principal

- Desenvolver uma aplicação web que recomende um jogo para o usuario.

## Para quem é essa aplicação

- Essa será uma aplicação util para pessoas que estão em busca de conhecer novos jogos ou que estejam em duvida de qual jogo jogar. 

# Funcionalidades principais

- Selecionar randomicamente entre uma lista filtrada pelo usuario.
- Devolver uma lista de jogos em que o usuario possa selecionar por si só.
- Filtrar jogos por notas, consoles, genero, etc.
- Integração com a api IGDB

# Tecnologias

- Linguagem: Python
- Framework: Flask
- Banco de dados: SQLite
- Frontend: HTML, CSS, JavaScript
- Outras ferramentas: Git, VS Code
